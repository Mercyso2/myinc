## Arquivo ENV

Use `.env.local`. O arquivo `.env.local.example` tem o modelo completo.

## Principais funções locais

- Planejamento mensal com categorias/formato.
- Produção em massa.
- Fazer tudo 100% automático.
- Gerar imagens em lote.
- Gerar Reels/Vídeos como roteiro + capa + storyboard.
- Carrossel com páginas e preview lateral.
- Cérebro IA com prompts base.
- Biblioteca com upload, referência aprovada/proibida, template, arquivar e excluir.
- Estúdio com melhorias rápidas: melhorar copy, mais premium, mais comercial, melhorar visual, revisor IA e aplicar template MYINC.
- Backup local.

## Observação Meta

Publicação real na Meta exige mídia em URL pública HTTPS. Localhost não é acessível pela Meta. Para publicar de verdade, configure Cloudinary, R2/S3, Supabase Storage público ou túnel HTTPS.
